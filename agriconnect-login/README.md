## üõ†Ô∏è Installation

1. **Clone the repo:**
   ```bash
   git clone <your-repo-url>
   cd agriconnect-login/agriconnect/agriconnect/agriconnect-login
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser:**
   [http://localhost:3000/dashboard](http://localhost:3000/dashboard)
